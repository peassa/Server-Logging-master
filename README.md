# Server Logging
A FiveM Plugin

# Installation
1. Download `Serverlogging`!
2. Install it into `/resources/`.
3. Enter your webhook url in `server.lua`
4. Make a Steam API Key in https://steamcommunity.com/dev/apikey and paste the key in `server.lua`
5. Start the plugin
6. Have fun!
